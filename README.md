## Como rodar o projeto

### **1** - Preenche o arquivo .env de acordo com o seu ambiente

### **2** - Instale as dependências do projeto

```
yarn
```

ou

```
npm install
```

### **3** - Rode o servidor de desenvolvimento

```
yarn dev
```

ou

```
npm run start
```
